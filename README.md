## <a name="local-development"> üñ•Ô∏è&nbsp;&nbsp; Local Development</a>

0.  **Prerequisites** <br>
    Make sure you have the following installed on your machine:

    - [Git](https://git-scm.com/)
    - [Node.js](https://nodejs.org/en) (v20+)
    - [pnpm](https://pnpm.io/) (Package Manager)

1.  **Clone the repository:**

    ```bash
    https://github.com/iamashishshah/kanban-project-from-cohort.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd kanban-project-from-cohort
    ```

3.  **Install dependencies:**

    ```bash
    pnpm install
    ```

4.  **Start the App:**

    ```bash
    pnpm dev
    ```

    Visit &nbsp;[http://localhost:3000](http://localhost:3000)&nbsp; to access your app.
